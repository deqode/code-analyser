# Install Plugins

## Clone the repo 
```
git clone https://github.com/deqode/CodeAnalyser
cd CodeAnalyser
```


## Install Plugin
- Install Nodejs Plugin
  ```
  bash installNodeModulesPlugin.sh
  ```
- Compile Go Binaries
  ```
  bash compileGoPluginBinaries.sh
  ```

## Ready
**Now you are ready to use Code Analyser**
